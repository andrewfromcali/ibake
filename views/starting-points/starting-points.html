Starting Points

Starting points are technologies people might use in a sentence like, "I know how to do it in [blank1] but I have no idea how it's done in [blank2]."

blank1 might be "java" and blank2 might be "ruby." Or, blank1 might be "hibernate" and blank2 might be "active record." Even though hibernate is a java technology, it's such a big complex peice of ORM software, it can be a starting point the same way the java language can be a starting point.
